0、IO

IO 即 input 、output.

  • Input 磁盘写入到内存 read()读进来
  • output 内存写到磁盘 write()写出去

1、阻塞和非阻塞是什么

  • 阻塞 不能同时满足多个请求，只有一个能满足，其他的需要等待
  • 非阻塞 就是可以满足多个

2、 同步和异步是什么？

  • 同步 就是串行

  • 异步 就是并行

    3、NIO

  • 同步非阻塞,实现方式就是 多路复用

  • 异步非阻塞，理论上最牛。但是实际上的使用效果没有那么好
